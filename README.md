# Itachi Sharingan Hand Sign Visualizer

An interactive web experience inspired by **Itachi Uchiha** from Naruto.

Using **MediaPipe Hands** and **Three.js**, the application detects hand gestures through your webcam and transforms them into cinematic particle-based visual effects.

## Features

- 👋 Real-time hand tracking
- 🔥 Fire Style (Tiger Seal)
- 🌙 Tsukuyomi visual effect
- 🌀 Genjutsu magic seal
- 👁 Mangekyō Sharingan animation
- 🎨 25,000-particle rendering engine
- 🎥 Webcam support
- 🖱 Manual control panel
- ✨ Bloom, glow and cinematic effects

## Technologies

- HTML5
- CSS3
- JavaScript (ES Modules)
- Three.js
- MediaPipe Hands
- WebGL

## Controls

| Gesture | Effect |
|----------|--------|
| Two Hands | Fire Style |
| Index + Middle | Tsukuyomi |
| Index Only | Genjutsu |
| Pinch | Mangekyō Sharingan |
| No Gesture | Neutral State |

You can also activate every technique from the control panel.

## How to Run

1. Download the repository.
2. Open `itachi.html` in a modern browser.
3. Allow camera permission.
4. Perform supported hand signs or use the manual controls.
